关于ImageView图片宽高比的问题

发表于 2017-05-02 | 分类于 Android | 阅读次数:

在Android的ImageView中图片宽高显示不友好，为了更明显地体现问题，给ImageView加了个chat_to_bg_normal.9.png（关于.9.png）作为background。

正常情况下，有些不是正方形的图片会显示得不是那么友好：

在加上android:adjustViewBounds=”true” 之后按照图片本身的轮廓友好显示：

AndroidDeveloper官网的文档截图：


意思是如果把这个属性设置为true的话，ImageView会通过调整边界值来保留图片的正常宽高比。
